# ATARS – Automated Time-Series Analysis and Reporting System

ATARS is an automation system that analyzes environmental time-series datasets and automatically generates statistical insights, charts, and a structured report.

The goal of ATARS is to reduce repetitive manual analysis by creating a reproducible pipeline that converts raw data into an interpretable report.

---

## What ATARS Produces

When the system runs on a dataset, it automatically generates:

• Statistical analysis of the dataset  
• Machine learning based anomaly detection  
• Time-series forecasting  
• Multiple analytical charts  
• A structured Word report summarizing all results  

The report includes:
- calculated statistical metrics
- anomaly detection results
- correlations between variables
- forecast outputs
- visual charts generated from the data

All results are generated automatically from the input dataset.

---

## Main Capabilities

- Automated statistical analysis  
- Isolation Forest anomaly detection  
- Holt-Winters forecasting  
- Chart generation for data insights  
- Automated report generation (.docx)  
- Optional LLM narrative generation with grounding validation
